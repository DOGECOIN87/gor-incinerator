# Cook Incinerator - Business Model

## Overview

Cook Incinerator is a premium token burning service for the Cookie Chain blockchain network. We provide the easiest way for users to reclaim rent from empty token accounts through a user-friendly web interface with Backpack wallet integration.

## Value Proposition

### For Users
- **Easiest Solution**: No command line required - simple web interface
- **Lowest Fees**: Only 5% service fee vs 15%+ from competitors
- **Backpack Integration**: Seamless connection with the only wallet supporting Cookie Chain
- **Professional Service**: >90% success rate with optimized transactions
- **Transparent Pricing**: All fees shown upfront before transaction
- **Safe & Secure**: Only closes zero-balance accounts

### Competitive Advantages
1. **Industry-Low Fees**: 5% vs competitors charging 15%+
2. **User Experience**: Web-based interface vs command line tools
3. **Wallet Integration**: Native Backpack wallet support
4. **Batch Processing**: Up to 14 accounts per transaction
5. **Proven Reliability**: >90% transaction success rate

## Revenue Model

### Service Fee Structure
- **5% of reclaimed rent** per transaction
- User keeps **95%** of reclaimed COOK
- No hidden fees or additional charges
- Transparent calculation shown before confirmation

### Example Transaction
```
Empty accounts found: 14
Total rent to reclaim: 0.0285 COOK
Service fee (5%): 0.00143 COOK
User receives: 0.0271 COOK
```

### Revenue Calculation
- Average rent per account: ~0.00203928 COOK
- Average transaction: 14 accounts = ~0.0285 COOK reclaimed
- Service fee per transaction: ~0.00143 COOK
- User receives: ~0.0271 COOK
