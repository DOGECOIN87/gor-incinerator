## Troubleshooting

### "Backpack wallet not found"
**Solution**: Install the Backpack wallet extension from [backpack.app](https://backpack.app)

### "Connection failed"
**Solution**: 
1. Make sure Backpack wallet is unlocked
2. Refresh the page and try again
3. Check that you're on the Gorbagana network

### "No empty accounts found"
**Solution**: Great! Your wallet is already optimized. No action needed.

### "Transaction failed"
**Solution**:
1. Make sure you have enough GOR for transaction fees
2. Try again - the service has automatic retry logic
3. Check your internet connection

### "Wallet disconnected"
**Solution**: Click "Connect Backpack Wallet" again to reconnect.
