# Design Update Summary

## Changes Made

The frontend has been redesigned to match the reference images with a dark green forest theme:

### Color Scheme
- **Background**: Dark forest green (#0d1f17 approximately)
- **Primary**: Medium green (#4ade80 / emerald-400)
- **Cards**: Slightly lighter green with transparency
- **Text**: Light gray/white for contrast

### Layout Changes

1. **Navigation**
   - Simplified header with "Gor Incinerator." branding
   - Minimal navigation links (Burn, About, Features, GitHub)
   - Connect wallet button on the right

2. **Hero Section**
   - Two-column layout (text left, visual right)
   - Large "Reclaim Your GOR Effortlessly" headline
   - Gradient text effect on "Effortlessly"
   - Large animated trash can icon (replacing the 3D model from reference)
   - Floating animation effect
   - Radial gradient glow behind the icon

3. **Stats Bar**
   - Horizontal stats section showing 5%, 95%, >90%
   - Clean, minimal design with green accents

4. **Burn Interface**
   - Centered card with backdrop blur
   - Green accent colors throughout
   - Simplified button styling

5. **Two-Column Info Section**
   - "Recent Activity" card on the left
   - "Why Gor Incinerator" card on the right
   - Matches the Events/Trending layout from reference

6. **Features Section**
   - Three-column grid
   - Icon cards with hover effects
   - Green accent colors

7. **Footer**
   - Minimal footer with branding and GitHub link

### Design Elements
- Backdrop blur effects on cards
- Subtle border colors
- Green gradient text
- Floating animation on hero icon
- Consistent spacing and typography
- Dark green theme throughout

## To Run

```bash
cd frontend
npm run dev
```

The design now closely matches the reference images with a dark green aesthetic, minimalist layout, and prominent trash can visual element.
